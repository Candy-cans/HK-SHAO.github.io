苏海老师的教导

  1. C语言是在演变的，C89是第一个完整的C语言标准，后继还有C99、C11等多个标准，其中C89是最为经典并且兼容性最好的，我们以后的课程中主要以C89为主进行学习，以防止C99的代码放到C89中就不行了。
  2. C语言是其他编程语言的基础，也是最为经典的编程语言，其地位无懈可击。所以我们从C语言开始学起。
  3. 当学会了C以后，其它编程语言也能够很自然的接受并学会使用，任何一门语言都是在不断发展的，语言就是拿来用的，达到目的即可。在精通之前，更重要的是学会使用。

t1 字符转换

这题需要了解的就是C语言中的字符型，也就是char数据类型。char字符对应了ASCII字符集中的字符，char类型占一个字节（byte），对照教材16面的单位转换 1 B = 8 Bit ，而 ，由于char的最高位用于校验，所以标准的ASCII码表（我们的第四版教材442面附录D）中总共有128个字符。实际能够在终端中显示的字符只有78个：

xxxxxxxxxx
123456789:;<=>?@ABCDEFGHIJKLMNOPQRSTUVWXYZ[\]^_`abcdefghijklmnopqrstuvwxyz{|}~

char类型在C语言中实质上是数字，例如 'A'==65 ，当它以 "%c" 形式输出时，显示为字符。char的范围是 -128 到 127，下面这张图显示了ASCII码的前128个字符（也是最常用的）。

对照码表，会发现大写英文字母和小写英文字母之间是等间距的，它们的间距是32，也就是 printf("%d", 'a'-'A'); 输出为32，所以 'A'+32 就是 'a' ，以此类推。这就实现了大写字母转换成小写字母，反之亦然。先根据字符序号判断字符是小写字母还是大写字母，然后用分支语句if(表达式){} 分别转换成小写或者大写，然后输出，就完成了本题。

xxxxxxxxxx
#include <stdio.h>
​
int main(void)
{
    char c = getchar();
    if (c >= 'A' && c <= 'Z')
        printf("%c", c + 'a' - 'A');
    else if (c >= 'a' && c <= 'z')
        printf("%c", c + 'A' - 'a');
    else
        printf("%d", c);
    return 0;
}

t2 反转三位数整数

这题需要了解C语言中的数组，char c; 是一个字符，而多个字符组成的序列就是字符串， char s[4]; 创建了一个容量为4的字符串。注意：使用 scanf 函数读入字符串的时候不需要 & 运算符（原因后文的Tips中会讲）。由于输入为三位数，可能带负号，所以最多也就4个字符。我们把输入的字符读进 s 中，然后判断第一个字符是否为负号，使用分支语句将数字逆序输出即可。

xxxxxxxxxx
#include <stdio.h>
​
int main(void)
{
    char s[4];
    scanf("%s", s);
    if (s[0] == '-')
        printf("%c%c%c", s[3], s[2], s[1]);
    else
        printf("%c%c%c", s[2], s[1], s[0]);
    return 0;
}

t4 一元二次方程求解

这题使用分支语句，再套用一元二次方程的求根公式即可。要注意的是，你需要使用 #include <math.h> 导入math库，使用 sqrt 函数计算 。

xxxxxxxxxx
#include <stdio.h>
#include <math.h>
​
int main(void)
{
    int a, b, c;
    float x1, x2, delta;
    scanf("%d%d%d", &a, &b, &c);
    delta = b * b - 4. * a * c;
    if (delta < 0.)
        printf("NO");
    else
    {
        x1 = (-b + sqrt(delta)) / (2. * a);
        x2 = (-b - sqrt(delta)) / (2. * a);
        printf("YES\n");
        printf("%f %f", x1, x2);
    }
    return 0;
}

t6 平方与立方

这题终于涉及了循环，for (int i = 0; i < n; i++) 无论是 C、C++、Java还是C#等，绝对是程序员最经常写到的的语句之一，循环内（有时也被称作迭代），常用的变量是 i,j,k ，这道题输入 ，就迭代 次，依次把 中的每个数字平方和立方输出即可。值得注意的是，由于题目限定了n必然是正数，所以我使用 unsigned 关键字表示整数n没有符号，也即它必然是正数。

xxxxxxxxxx
#include <stdio.h>
​
int main(void)
{
    unsigned int n;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++)
        printf("%d %d\n", i * i, i * i * i);
    return 0;
}

注意，上述代码并没有遵循C89标准。C89要求所有的变量必须在第一条可执行语句之前定义，而上述代码在 for 循环中定义了整型 i ，这种方法在C99及以后的新标准中才被允许，然而像 for (int i = 0; i < n; i++) 这种用法在 C++、Java、C#等语言中太过常见，所以这里及以后涉及了 for 循环的代码中，我并不打算遵循C89标准。

t10 找出车牌号

这题需要了解的是取一个整数的各个数位的方法。取数位的方法是 x/n%10 ，以取 的百位数为例。

C语言会将其自动转换为整数，于是我们得到了 的百位 。

题目要求四位数是平方数，然而我们没有一个较好的方法判断一个数是否为平方数，我们可以考虑生成所有的四位平方数，然后依次判断。注意到 和 ，于是我们迭代生成四位平方数，然后取各位数字，依照题意判断，满足就输出即可。

xxxxxxxxxx
#include <stdio.h>
​
int main(void)
{
    for (int i = 32; i < 100; i++)
    {
        int x1, x2, x3, x4;
        int n = i * i;
        x1 = n / 1000 % 10;
        x2 = n / 100 % 10;
        x3 = n / 10 % 10;
        x4 = n / 1 % 10;
        if (x1 == x2 && x3 == x4 && x1 != x3)
            printf("%d", n);
    }
    return 0;
}

t16 计算月份的天数

这题需要了解判断闰年的方法。考虑到数组的起始下标为0，为了使代码简洁，我们可以把12个月的天数存进一个长度为13的数组，day[0] 为0占位，然后剩余的数对应月份的天数。2月通常为28天，考虑到闰年2月为29天，这里我们需要进行判断。

t17 计算数的和

这题需要注意的是无限循环，也即每一次循环都要输入一个数字，直到输入数字0就跳出循环。无限循环的方式有两种，一种是 while(1){} ，另一种 for(;;){} 更简洁也更常用。注意要使用 break; 跳出循环，如果没有 break; ，程序就会陷入死循环。

xxxxxxxxxx
#include <stdio.h>
​
int main(void)
{
    int n, i;
    for (;;)
    {
        scanf("%d", &i);
        if (i > 0)
            n += i;
        else if (i == 0)
        {
            printf("%d", n);
            break;
        }
    }
    return 0;
}

t18 计算人数

注意到男人数和小孩数是升序的，我们从男人数和小孩数为0开始迭代， 计算出剩余的先令，如果能被2整除，也即 a%2==0 则 a/2 为女人数，再符合人数之和等于30，也即 就输出答案。

xxxxxxxxxx
#include <stdio.h>
​
int main(void)
{
    int n = 30, m = 50;
    for (int i = 0; i <= n / 3; i++)
        for (int j = 0; j < n; j++)
        {
            int a = m - 3 * i - j;
            if (a % 2 == 0 && i + a / 2 + j == n)
                printf("%d %d %d\n", i, a / 2, j);
        }
    return 0;
}

​

Tips

常见的语法错误

  • 漏掉括号 ( 或者 ) 导致括号不匹配，无法通过编译
  • 不能把全角符号和半角符号混用 ;； ()（） 是有明显区别的，写代码时要使用英文输入
  • 语句结尾漏掉 ; 或者多余的 ; 导致编译失败或者运行错误
  • 分支语句 if 后写多余的分号，或者忘记写 else
  • 将逻辑运算符写错，例如 && 写成 & 、|| 写成 |
  • for 循环分号写错，是 for(;;) 而不是 for(,,)
  • C语言严格区分字符和字符串，所以 '' 和 "" 不能像 Python、JavaScript 那样混用
  • 关系运算 == 写成 = 导致运算错误（要注意C语言中 = 是赋值，而 == 才是判断是否值相等）
  • 使用 scanf 函数输入时，变量前忘记写 & ，例如 int n;scanf("%d", n); 是错误的

scanf 输入问题

scanf 函数的参数是字符串常量和变量的地址， n 代表的是变量n的值， & 是取地址运算符， &n 代表了变量n的地址，于是 scanf("%d", &n); 语句表示的是：先获取变量n的地址，然后把输入的内容转换为整型，存放到相应地址中，也即为变量n赋值。

当定义一个数组时，例如定义数组 char s[4]; ，此时 s 代表的是 s[0] 的地址，即 s==&s[0] ，而 s[0],s[1],s[2],s[3] 才是变量 s 的值。所以输入字符串时 scanf("%s", &s); 是错的（有时编译器也能够通过编译并正常运行，但是会给出警告），而 scanf("%s", s); 才是对的。
